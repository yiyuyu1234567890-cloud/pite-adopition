# 暖爪宠物收养网站

一个静态宠物收养展示网站，包含宠物筛选、搜索、收藏、详情弹窗和收养申请表单。

## 本地预览

双击 `start-website.bat`，然后打开：

```text
http://127.0.0.1:5173/
```

## GitHub Pages 部署

1. 在 GitHub 新建一个仓库。
2. 上传本文件夹中的所有文件。
3. 进入仓库的 `Settings`。
4. 打开 `Pages`。
5. `Build and deployment` 选择 `Deploy from a branch`。
6. `Branch` 选择 `main`，目录选择 `/root`。
7. 保存后等待部署完成。
